# Hooks

Custom React hooks para lógica reutilizable.

## Hooks sugeridos:

- **`useCart.ts`** - Gestión del carrito de compras
- **`useAuth.ts`** - Autenticación y usuario actual
- **`useProducts.ts`** - Fetch y filtrado de productos
- **`useWishlist.ts`** - Lista de deseos
- **`useNotifications.ts`** - Notificaciones del usuario
- **`useChat.ts`** - Chatbot de IA
- **`useSizeConverter.ts`** - Conversión de tallas
- **`useDebounce.ts`** - Debounce para búsquedas
