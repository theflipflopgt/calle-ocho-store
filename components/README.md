# Components

Componentes reutilizables de React organizados por funcionalidad.

## Estructura:

- **`ui/`** - Componentes UI básicos (Button, Input, Card, Modal, etc.)
- **`products/`** - Componentes relacionados con productos
  - ProductCard, ProductGrid, ProductDetail
  - ColorPicker, SizeSelector
  - ProductImages
- **`cart/`** - Componentes del carrito de compras
  - CartItem, CartSummary, CartDrawer
- **`layout/`** - Componentes de layout
  - Header, Footer, Navbar
  - Sidebar, MainLayout
- **`ai/`** - Componentes del chatbot de IA
  - ChatWidget, ChatMessage, ChatInput
- **`admin/`** - Componentes del panel de administración
  - ProductForm, OrderList, StatsCard
